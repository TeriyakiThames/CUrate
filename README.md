# Curate

CUrate is a website made to help students in Chula find what restaurants they want to eat in.

## Tech Stack

- Next JS
- Tailwind CSS
- Firebase

## Getting Started

- Run `pnpm i` to install dependencies.
- Run `pnpm run dev` to start a dev server.
- Add a new branch from `development` and name it `feat/name_of_addition` when adding something.
- Don't push code into `main` directly, go through `development` first -> then `main`.
